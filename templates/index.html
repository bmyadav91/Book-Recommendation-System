Clear Your Doubts by FAQs

This book recommendation system operates based on your interests in different genres. If you click on any book and then return to the homepage, you will see book recommendations tailored to your preferences. Additionally, if you search for a specific book, the system will suggest relevant titles. All of these features are powered by machine learning algorithms.

When you upload a new dataset, the system will automatically store the data in the database, retrain the model based on your data, and create a .pkl file. This file will be uploaded to an S3 bucket for easy access, eliminating the need for frequent model retraining.

Click the link or icon to visit GitHub and read the README.md file for setup instructions.

All documentation like Complete documentation for the project, including HLD, LLD, and architecture, is available in the docs folder of the GitHub repository.
